This folder contains curated guidance for agent use:

- `project_map.md` describes how the Python API + CLI interact with job definitions.
- `schema.json` captures the JSON format of bundles for validation/reference.
- `examples/` holds reusable shell scripts illustrating common submission patterns.
- `cli_reference.md` lists CLI commands and flags relevant to agents.
- `resource_rules.md` documents SLURM resource expectations.
